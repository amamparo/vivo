Ableton Live Setup

  1. Open Ableton Live
  2. Go to Preferences (Cmd+,)
  3. Select the Link, Tempo & MIDI tab
  4. Under Control Surface, select LiveEarsOSC
  5. Leave MIDI Input and Output set to None

LiveEarsOSC is automatically installed and updated by Live Ears.
You only need to select it once in Ableton's preferences.
